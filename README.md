# Landing-Page-Banda-de-Rock

Landing Page – Rock Band (Events and Promotion)

This landing page was created and designed for a rock band looking to promote its identity, music, and upcoming events. It conveys the band’s energy, personality, and style, effectively capturing the attention of potential concertgoers.

The technologies used in this project were HTML and CSS, where we utilized them as core front-end technologies: HTML handles the structure and semantics, while CSS defines the visual appearance and responsive design.
HTML is the standard markup language used to create the skeleton and semantic structure of a webpage. It defines building blocks such as headings, paragraphs, lists, images, and links.
CSS is the style sheet language applied to HTML documents to define the visual presentation: colors, fonts, margins, borders, and the positioning of elements.

PROJECT STRUCTURE.
Initially, we see the rock band’s name and a navigation bar containing options such as Events, About Us, Music, Contact, and Gallery. Next is a section featuring Tour/Dates: an updated concert calendar, links to buy tickets, studio and live photos, music videos, contact information, and a booking form. The design is dark and minimalist.
